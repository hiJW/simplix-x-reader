# SimpliX Userscript

Tampermonkey/Violentmonkey userscript version of SimpliX.

## Install

1. Install Tampermonkey or Violentmonkey.
2. Create a new userscript.
3. Copy the contents of `simplix.user.js`.
4. Save the script and open `x.com` / `twitter.com`.
5. Use the userscript manager menu to set your DeepSeek API key.

For mobile use, Firefox + Violentmonkey is recommended.
